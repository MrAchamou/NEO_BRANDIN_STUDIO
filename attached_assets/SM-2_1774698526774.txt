# INSTRUCTION POUR REPLIT AI

## Module 01.2 — Palette Generator

### Objectif

Modifier l'application Neo Branding Studio pour implémenter un système de génération de palettes de couleurs **ultra-précises** pour le sous-module Palette Generator, en utilisant l'exemple parfait fourni comme **golden reference**.

L'objectif est de garantir que **chaque génération** respecte exactement la structure, le format de sortie (tableaux + JSON), les calculs WCAG et les contraintes d'accessibilité de l'exemple, quels que soient les paramètres d'entrée.

---

## 1. Contexte

**Fichier actuel à modifier** :  
`api-server/src/routes/openai/enhance-prompts.ts` (section Module 01)

**Données d'entrée du formulaire** :
```typescript
interface BrandBrief {
  brandName: string;      // ex: "Theravectys"
  sector: string;         // ex: "tech", "luxury", "streetwear", "fitness", "cosmetic"
  tone: string;           // ex: "professionnel", "audacieux", "minimaliste"
  values: string[];       // ex: ["innovation", "rigueur", "fiabilité"]
  primaryColorHint?: string;  // optionnel: si l'utilisateur propose une couleur
}
```

**Sortie attendue** : Un prompt structuré qui générera une palette complète avec tableaux WCAG et JSON.

### 2.3. Fichier : `api-server/src/prompts/modules/module-01-2-palette/prompt-builder.ts`

Ce fichier construit le prompt final en remplaçant les parties variables de l'exemple parfait.

```typescript
import { goldenExample } from './golden-example.json';
import { sectorPalettes, wcagRequirements } from './mapping-rules.json';

interface BrandBrief {
  brandName: string;
  sector: string;
  tone: string;
  values: string[];
  primaryColorHint?: string;
}

interface ColorPalette {
  primary: { hex: string; rgb: string; meaning: string; role: string };
  secondary: { hex: string; rgb: string; meaning: string; harmony: string };
  accent: { hex: string; rgb: string; meaning: string };
  neutrals: {
    '01': string;
    '02': string;
    '03': string;
    '04': string;
    '05': string;
  };
}

export function buildPalettePrompt(brief: BrandBrief): string {
  // Récupérer la palette selon le secteur
  const sectorPalette = sectorPalettes[brief.sector] || sectorPalettes.tech;
  
  // Adapter la palette si l'utilisateur a fourni une couleur primaire
  let primaryColor = sectorPalette.primary;
  if (brief.primaryColorHint) {
    primaryColor = {
      ...primaryColor,
      hex: brief.primaryColorHint,
      rgb: hexToRgb(brief.primaryColorHint)
    };
  }
  
  // Construire la description des valeurs
  const valuesText = brief.values.join(', ');
  
  // Adapter la signification psychologique selon le ton
  const toneModifier = getToneModifier(brief.tone);
  
  // Construire le prompt final
  let prompt = goldenExample.goldenExample;
  
  // Remplacer les variables de base
  prompt = prompt.replace(/Theravectys/g, brief.brandName);
  prompt = prompt.replace(/secteur tech/g, `secteur ${brief.sector}`);
  prompt = prompt.replace(/ton professionnel/g, `ton ${brief.tone}`);
  prompt = prompt.replace(/innovation, rigueur scientifique, fiabilité, expertise, engagement/g, valuesText);
  
  // Remplacer la section Couleur primaire
  const primarySection = `
1) Couleur primaire (60% usage)
- Propose 1 couleur primaire "${getSectorColorDescription(brief.sector)}" avec code HEX + RGB.
- Donne sa signification psychologique (${primaryColor.meaning}${toneModifier}) et son rôle (${primaryColor.role}).
- HEX: ${primaryColor.hex} | RGB: ${primaryColor.rgb}
`;
  
  // Remplacer la section Couleur secondaire
  const secondarySection = `
2) Couleur secondaire (30% usage)
- Propose 1 couleur secondaire complémentaire/harmonique avec code HEX + RGB.
- HEX: ${sectorPalette.secondary.hex} | RGB: ${sectorPalette.secondary.rgb}
- Explique l'accord colorimétrique: ${sectorPalette.secondary.harmony}
`;
  
  // Remplacer la section Couleur accent
  const accentSection = `
3) Couleur accent / CTA (10% usage)
- Propose 1 couleur accent orientée conversion (CTA) avec code HEX + RGB.
- HEX: ${sectorPalette.accent.hex} | RGB: ${sectorPalette.accent.rgb}
- Décris l'émotion générée: ${sectorPalette.accent.meaning}
- Doit ressortir sur la primaire + sur fond blanc.
`;
  
  // Remplacer la section Neutres
  const neutralsSection = `
4) 5 nuances neutres (HEX exacts)
- Neutral-01 (background très clair): ${sectorPalette.neutrals['01']}
- Neutral-02 (surface / cards): ${sectorPalette.neutrals['02']}
- Neutral-03 (bordures / separators): ${sectorPalette.neutrals['03']}
- Neutral-04 (texte secondaire): ${sectorPalette.neutrals['04']}
- Neutral-05 (texte principal / fond sombre): ${sectorPalette.neutrals['05']}
`;
  
  // Remplacer la section WCAG avec calculs précis
  const wcagSection = `
5) Validation WCAG 2.1 AA (calculée)
- Calcule et affiche les contrast ratios (min ${wcagRequirements.normalText}:1 pour texte normal, ${wcagRequirements.largeText}:1 pour large).
- Utilise la formule: ${wcagRequirements.formula}
- Donne un tableau clair "VALIDES / INVALIDES" pour les paires suivantes:

| Paire | Ratio | Statut |
|-------|-------|--------|
| Texte Neutral-05 sur blanc | [calculer] | [valider] |
| Texte Neutral-04 sur blanc | [calculer] | [valider] |
| CTA (${sectorPalette.accent.hex}) sur blanc | [calculer] | [valider] |
| Texte #FFFFFF sur primaire (${primaryColor.hex}) | [calculer] | [valider] |
| Texte #FFFFFF sur secondaire (${sectorPalette.secondary.hex}) | [calculer] | [valider] |
| Texte #FFFFFF sur CTA (${sectorPalette.accent.hex}) | [calculer] | [valider] |
| Texte Neutral-04 sur Neutral-02 (${sectorPalette.neutrals['04']} sur ${sectorPalette.neutrals['02']}) | [calculer] | [valider] |
| Texte Neutral-01 sur Neutral-05 (${sectorPalette.neutrals['01']} sur ${sectorPalette.neutrals['05']}) | [calculer] | [valider] |

- Si une paire est invalide: propose un ajustement en itérations (max 3) en lightening/darkening (indiquer X%), en préservant la teinte (hue) ±3° et en limitant la variation de saturation (±10%). Fournis le NOUVEAU HEX + RGB + nouveau ratio.
`;
  
  // Remplacer la section Applications
  const applicationsSection = `
6) Applications recommandées (règles d'usage)
- Boutons: primary = ${primaryColor.hex}, secondary = ${sectorPalette.secondary.hex}, ghost = transparent avec bordure ${sectorPalette.neutrals['03']}
- Hover states: primary darken 10%, secondary lighten 10%
- Liens: ${sectorPalette.accent.hex} avec underline on hover
- Fonds: Neutral-01 pour fond principal, Neutral-02 pour cards
- Textes: H1/H2 = ${primaryColor.hex} sur fond clair, body = Neutral-05
- Icônes: ${sectorPalette.secondary.hex} pour les actions secondaires
- Badges: success = #10B981, info = ${sectorPalette.accent.hex}, warning = #F59E0B, error = #EF4444
- Print (CMYK approx): ${hexToCmyk(primaryColor.hex)}, ${hexToCmyk(sectorPalette.secondary.hex)}
`;
  
  // Remplacer la section Palette saisonnière
  const seasonalSection = `
7) Palette saisonnière / mood board
- Propose 1 mini-variation "Conference / Lab" dérivée de la palette principale:
  * ${adjustColor(sectorPalette.primary.hex, -20)} (primary plus profond)
  * ${adjustColor(sectorPalette.accent.hex, 15)} (accent plus lumineux)
  * ${sectorPalette.neutrals['03']} (neutral intermédiaire)
- Usage: supports de conférence, présentations, documents institutionnels
- Cohérence: conserve l'identité de marque tout en apportant de la profondeur
`;
  
  // Assembler le prompt final
  prompt = prompt.replace(/1\) Couleur primaire \(60% usage\).*?(?=2\) Couleur secondaire)/s, primarySection);
  prompt = prompt.replace(/2\) Couleur secondaire \(30% usage\).*?(?=3\) Couleur accent)/s, secondarySection);
  prompt = prompt.replace(/3\) Couleur accent \/ CTA \(10% usage\).*?(?=4\) 5 nuances neutres)/s, accentSection);
  prompt = prompt.replace(/4\) 5 nuances neutres \(HEX exacts\).*?(?=5\) Validation WCAG)/s, neutralsSection);
  prompt = prompt.replace(/5\) Validation WCAG 2\.1 AA \(calculée\).*?(?=6\) Applications recommandées)/s, wcagSection);
  prompt = prompt.replace(/6\) Applications recommandées.*?(?=7\) Palette saisonnière)/s, applicationsSection);
  prompt = prompt.replace(/7\) Palette saisonnière.*?(?=\[PARAMÈTRES TECHNIQUES\])/s, seasonalSection);
  
  // Ajouter l'instruction JSON final
  prompt += `

**JSON FINAL OBLIGATOIRE** (machine-readable):
\`\`\`json
{
  "brand": "${brief.brandName}",
  "sector": "${brief.sector}",
  "palette": {
    "primary": { "hex": "${primaryColor.hex}", "rgb": "${primaryColor.rgb}", "usage": "60%" },
    "secondary": { "hex": "${sectorPalette.secondary.hex}", "rgb": "${sectorPalette.secondary.rgb}", "usage": "30%" },
    "accent": { "hex": "${sectorPalette.accent.hex}", "rgb": "${sectorPalette.accent.rgb}", "usage": "10%" },
    "neutrals": {
      "neutral-01": "${sectorPalette.neutrals['01']}",
      "neutral-02": "${sectorPalette.neutrals['02']}",
      "neutral-03": "${sectorPalette.neutrals['03']}",
      "neutral-04": "${sectorPalette.neutrals['04']}",
      "neutral-05": "${sectorPalette.neutrals['05']}"
    }
  },
  "wcag": {
    "standard": "WCAG 2.1 AA",
    "normalTextMinRatio": ${wcagRequirements.normalText},
    "largeTextMinRatio": ${wcagRequirements.largeText}
  }
}
\`\`\`
`;
  
  return prompt;
}

// Fonctions utilitaires
function hexToRgb(hex: string): string {
  const r = parseInt(hex.slice(1, 3), 16);
  const g = parseInt(hex.slice(3, 5), 16);
  const b = parseInt(hex.slice(5, 7), 16);
  return `${r},${g},${b}`;
}

function hexToCmyk(hex: string): string {
  // Conversion simplifiée pour le prompt
  const r = parseInt(hex.slice(1, 3), 16) / 255;
  const g = parseInt(hex.slice(3, 5), 16) / 255;
  const b = parseInt(hex.slice(5, 7), 16) / 255;
  
  const k = 1 - Math.max(r, g, b);
  const c = (1 - r - k) / (1 - k);
  const m = (1 - g - k) / (1 - k);
  const y = (1 - b - k) / (1 - k);
  
  return `${Math.round(c * 100)}%, ${Math.round(m * 100)}%, ${Math.round(y * 100)}%, ${Math.round(k * 100)}%`;
}

function adjustColor(hex: string, percent: number): string {
  // Simplifié pour le prompt - la vraie implémentation serait plus complexe
  return percent > 0 ? `${hex} (${percent}% plus clair)` : `${hex} (${Math.abs(percent)}% plus foncé)`;
}

function getSectorColorDescription(sector: string): string {
  const descriptions: Record<string, string> = {
    tech: "scientific tech (bleu/bleu-vert ou équivalent crédible)",
    luxury: "premium heritage (bleu profond, or, tons nobles)",
    streetwear: "urban authentic (noir profond, rouge audacieux)",
    fitness: "performance energy (bleu électrique, rouge dynamique)",
    cosmetic: "natural elegance (tons doux, organiques)"
  };
  return descriptions[sector] || descriptions.tech;
}

function getToneModifier(tone: string): string {
  const modifiers: Record<string, string> = {
    professionnel: ", avec une approche maîtrisée et experte",
    audacieux: ", avec une touche d'innovation disruptive",
    minimaliste: ", dans une approche épurée et essentielle"
  };
  return modifiers[tone] || "";
}
```
